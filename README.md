# SDEN
Basic 3D Framework Runs in OpenGL On Windows, Linux and Mac.

How to build
---------

Windows, Linux, Mac:

	mkdir build
	cd build
	cmake -Wno-dev ..
	make
	./SDEN_Project

Visual studio:

	mkdir build
	cd build
	cmake ..
	now open the visual studio project
